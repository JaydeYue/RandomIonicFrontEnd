# Setup

Using Ionic and Node.js for the backend. Make sure you have them installed

1. **Install Required libraries**

```bash
$ npm install
$ npm install -g cordova ionic
```

3. **Running locally**

```bash
$ ionic serve
```

Only tab 1 is in use, the other two tabs are placeholder.
